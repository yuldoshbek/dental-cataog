# Контекст Проекта
**Цель проекта**: Создать внутренний web-каталог (шпаргалку) стоматологического оборудования для менеджеров по продажам.
**Стек технологий**: React, Vite, Tailwind CSS, Lucide React.

**Основные правила и требования**:
1. Каталог предназначен для менеджеров (подсказка во время звонков), а не для клиентов напряму. Это "шпаргалка", помогающая строить диалог.
2. Цены будут использоваться примерные (рыночные) для старта диалога, с указанием ценовых диапазонов по странам производства. Точные коммерческие цены будут добавлены позже.
3. Данные по оборудованию структурируются по категориям (установки, компрессоры, автоклавы и т.д.).
4. Каждое крупное архитектурное решение и добавление новых особенностей фиксируется в changelog.md.
5. Инструмент должен позволять менеджеру открыть панель и "сразу видеть" всё, что может предложить клинике по запросу. В один клик можно будет скопировать данные для отправки в мессенджер.
